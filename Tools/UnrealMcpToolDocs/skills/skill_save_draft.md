# unreal.skill_save_draft

**Category**: skills
**Title**: Save Skill Draft
**Risk level**: medium

Writes edited skill draft content under Saved/UnrealMcp before optional promotion.

## Capabilities

- Requires write: true
- Requires build: false
- Requires external process: false
- Requires restart: false
- Requires lock: false
- Dry-run support: false
- Preflight support: true
- Postcheck support: true
- Test coverage: core

## Input schema

```json
{
  "type": "object",
  "properties": {
    "skillName": {
      "type": "string",
      "description": "Draft skill folder name."
    },
    "title": {
      "type": "string",
      "description": "Draft title used when draftText is empty."
    },
    "goal": {
      "type": "string",
      "description": "Draft goal used when draftText is empty."
    },
    "summary": {
      "type": "string",
      "description": "Draft summary used when draftText is empty."
    },
    "draftText": {
      "type": "string",
      "description": "Complete SKILL.md text to save as a draft."
    },
    "overwrite": {
      "type": "boolean",
      "description": "Overwrite an existing draft.",
      "default": true
    }
  },
  "required": [],
  "additionalProperties": false
}
```
